Bluetooth Light Switcher

The Aim

The task of this project was simple: turn on or off a light. I wanted to be able to do this remotely using my phone.


The Implementation

Using magnets, I attached a small cardboard box on top of the light switch plate. The box provided an opening for the switch to go in. Inside the box was an Arduino with two servos, each one used to flip the switch in a certain direction. An LED light was visible through the box to indicate a signal being received from the Android phone.
The sequence looks like this:

Phone sends signal
↓
Bluetooth module receives signal and passes it to Arduino
↓
Arduino analyses signal
↓
Arduino sends instructions to the LED and servos.


Materials Used

Arduino Uno
Arduino Bluetooth Module
Two Micro Servos
BlueTerm (This app is open source!)
Any Android device


Future Expansions

I think it would be really neat to expand this project by
1. Instead of using Bluetooth signals from an Android, using signal processing to control the light through whistling
2. Instead of using an Arduino, use a PIC to make it much smaller
3. Make more things to automate a dorm room (blinds, thermostat, etc.)
